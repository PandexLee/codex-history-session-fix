# Security Policy

## Scope

This repository contains an unofficial workaround that can repackage and reinstall a local Windows MSIX application package. Treat the scripts as high-impact local system automation.

## Before Running

- Read the script you plan to run.
- Prefer `-DryRun` first.
- Do not run scripts from a fork or copy you do not trust.
- Do not paste verification output publicly without checking for local paths or identifiers.

## Reporting Security Issues

Open a GitHub issue with a minimal description and avoid posting secrets, tokens, local database contents, thread titles, prompts, or private file paths.

If the issue involves a real secret or private data exposure, do not include the secret itself. Describe where the exposure occurs and how to reproduce it with placeholders.
